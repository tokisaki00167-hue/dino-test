# DINO 自動化測試練習（Python + Playwright）

**專案簡介**  
這是一個以 Chrome 小恐龍遊戲（Dino Game）為場景的自動化測試練習專案，目標是練習測試流程（規劃 → 執行 → 紀錄）與自動化腳本的撰寫。

**主要功能**
- 自動開啟 Dino 遊戲並開始遊戲流程
- 讀取遊戲內部狀態（速度、障礙物位置）來判斷跳躍時機
- 合併黏在一起的障礙物（多顆仙人掌）作為一段進行判斷
- 終端即時輸出速度、跳躍閾值與最近障礙物距離
- 記錄並列出極端/例外事件
